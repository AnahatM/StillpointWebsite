# Screenshots

Drop the real Stillpoint app screenshots in this folder, then wire them up by
adding a `src` (and `alt`) to the matching `<ScreenshotFrame>` component.

Suggested files (any names work — just match them in the components):

| File | Used in | Suggested content |
| --- | --- | --- |
| `dashboard.png` | `src/components/Hero.tsx` | Main dashboard with live posture score + skeleton overlay |
| `focus.png` | `src/components/Features.tsx` | Pomodoro timer / tasks / sticky notes |
| `analytics.png` | `src/components/Features.tsx` | Analytics view with the trend chart |

Recommended: 16:10 / 16:9 PNGs at roughly 1600px wide for crisp rendering.
For the best look, capture against the **Midnight** theme so the screenshots
blend into the site's navy palette.
